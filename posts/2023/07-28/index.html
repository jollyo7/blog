ブログ関連作業 content_copy

ブログそのものじゃない部分で行った作業をメモしておく。

Feed

  • <head> タグ内の link に Feed URL を追加

各リーダーに(Feed URL ではなく)サイトの URL を入れるだけで登録できるようになった。

Google アナリティクス

  • 登録後、指定されたトラッキングコードを <head> タグ内に記載

アクセスログの代わりくらいにはなるはず。

Google Search Console

  • 登録後、指定された html ファイルを設置してサイト所有権の確認
    • Google アナリティクス用のコードだけでも確認出来たが、複数経路あったほうがよいとのことなので

サイト内検索が動くようになったはず。

textlint

  • Node.js/npm/n を入れて npm からインストール
  • VScode の拡張 vscode-textlint を入れる事で、通常の linter と同じ動きをするように
  • 使う textlint プラグインやパラメータは使いながら修正していく
